# Bootstrap
This project was developed as part of an internship task to explore Bootstrap 5 and design clean, modern, and responsive web pages. The focus of the project is on understanding Bootstrap components and combining them creatively to build complete pages instead of directly copy-pasting example templates.
The project includes three fully responsive pages:
Home Page
About / Services Page
Contact Page with a form
Technologies Used
HTML5
CSS3
Bootstrap 5

Identified reusable UI components such as navbar, cards, grid system, hero sections, and forms.
Designed the Home page using a navbar, hero section, feature cards, and footer.
